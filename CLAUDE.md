# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a documentation handbook collecting popular GitHub open-source solutions for file parsing across multiple formats:
- PDF parsing and extraction
- Word document processing
- Excel/spreadsheet parsing
- Scanned document OCR
- Image text extraction (including scanned documents)
- Plain text processing
- Layout analysis techniques
- Text and table extraction practices

## Repository Structure

```
docs/                    # Core documentation (overview, getting started, best practices)
parsing-guides/          # Parsing guides organized by file type
  ├── pdf/              # PDF parsing (tables, layout, OCR)
  ├── word/             # Word document processing
  ├── excel/            # Excel/spreadsheet parsing
  ├── ppt/              # PowerPoint content extraction
  ├── images/           # Image OCR, scanned document processing
  ├── markdown/         # Markdown parsing, format conversion
  ├── html/             # HTML content extraction, parsing
  ├── xml/              # XML parsing, data extraction
  ├── yaml/             # YAML configuration parsing
  ├── rtf/              # Rich Text Format parsing
  ├── email/            # Email (EML/MSG) extraction
  ├── archive/          # Archive (ZIP/TAR) parsing
  └── plain-text/       # CSV, JSON, TXT processing
tool-reviews/            # Tool/library comparisons and reviews
common-patterns/         # Reusable patterns (error handling, memory optimization)
examples/                # Comprehensive examples (invoice parser, resume parser, etc.)
```
