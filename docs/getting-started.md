# 快速开始

## 环境准备

### Python 环境（推荐）

```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# 安装基础依赖
pip install -r requirements.txt
```

### Node.js 环境

```bash
# 安装依赖
npm install
```

## 运行示例

### Python 示例

```bash
# 进入示例目录
cd parsing-guides/pdf/examples

# 运行示例
python extract-text.py
```

### Node.js 示例

```bash
# 运行示例
node parsing-guides/pdf/examples/extract-text.js
```
